Options UI

UI

Fine Uploader UI mode has a few different options as well as some options pertaining specifically to the UI which Core mode does not have.

Any options that exist in Core mode also exist in UI mode, and, in most cases, can be overridden.

Option Name

Type

Default

Description

element

String

null

Container for the default drop zone

listElement

String

null

Container for the item list.

multiple

Boolean

true

When true this will prevent the user from simultaneously selecting or dropping more than one item. Dropping or selecting another item will clear the upload list. If another is already uploading, it will be cancelled. To ignore rather than cancel, simply return false in the ‘validate’ or ‘submit’ event handlers.

deleteFile option

Note:

deleteFile is also in the Core mode options. This section defines UI specific options for deleteFile

Option Name

Type

Default

Description

confirmMessage

String

'Are you sure you want to delete {filename}?'

The message displayed in the confirm delete dialog.

deletingFailedText

String

'Delete failed'

The status message to appear next to a file that is pending deletion.

deletingStatusText

String

'Deleting...'

The status message to appear next to a file that has failed to delete.

forceConfirm

Boolean

false

display option

Option Name

Type

Default

Description

fileSizeOnSubmit

Boolean

false

Enable or disable the display of the file size next to the file after it has been submitted.

prependFiles

Boolean

false

When true batches of files are added to the top of the UI’s file list. The default is to append file(s) to the bottom of the list.

dragAndDrop option

Option Name

Type

Default

Description

extraDropzones

Array

[]

Designate additional drop zones for file input.

hideDropzones

Boolean

true

Use if you would like dropzone elements to be hidden or not.

disableDefaultDropzone

Boolean

false

Use if you are contributing your own dropzone(s) that is/are not the default one.

editFilename option

Option Name

Type

Default

Description

enabled

Boolean

false

Enable or disable the user’s ability to edit the names of submitted files before upload.

failedUploadTextDisplay option

Option Name

Type

Default

Description

enableTooltip

Boolean

true

Enable or disable a tooltip that will display the full contents of the error message when the mouse pointer hovers over the failed item.

maxChars

Number

50

The maximum amount of characters to display next to each item. Ignored unless mode is 'custom'

mode

String

'default'

Set the message to display next to each failed file. One of: 'default' which displays the failedUploadText, 'custom' which displays the error response from the server, or 'none' which displays no text.

responseProperty

String

'error'

The property from the server response that contains the error text to display next to a failed item. Ignored unless mode is 'custom'

messages option

Note:

messages is also in the Core mode options. This section defines UI specific options for messages

Option Name

Type

Default

Description

tooManyFilesError

String

You may only drop one file.

Text sent to showMessage in the ‘error’ event handler when multiple is false and more than one file is dropped at once.

unsupportedBrowser

String

Unrecoverable error - the browser does not permit uploading of any kind.

Text displayed to users who have ancient browsers.

retry option

Option Name

Type

Default

Description

autoRetryNote

String

'Retrying {retryNum}/{maxAuto} ...'

The text of the note that will optionally appear next to the item during automatic retry attempts. Ignored if showAutoRetryNote is false.

showButton

Boolean

false

Enable or disable the showing of a button/link nex to the failed item after all retry attempts have been exchausted. Clicking the button/link will force the uploader to make another attempt.

showAutoRetryNote

Boolean

true

Enable or disable a status message appearing next to the item during auto retry attempts.

Option Name

Type

Default

Description

Boolean

false

Enable or disable the usage of showPrompt by Fine Uploader to prompt the user for a filename for a pasted file.

namePromptMessage

String

'Please name this image'

Text that will appear in the showPrompt dialog.
