5.0.8

Attention

iOS 8.0.0 contains some serious bugs that prevent uploading in some cases. Workarounds are included in Fine Uploader in an attempt to deal with these issues. Unfortunately, there are no known workarounds for iOS 8.0.0 Safari, and uploading in that browser is not possible at this time Please see the blog post on this topic for more details.

Note

Version 5.0 brings some breaking changes. See the upgrading to 5.x page for help on upgrading from a 4.x version.
